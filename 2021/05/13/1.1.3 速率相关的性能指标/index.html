    速率相关的性能指标

    brlink
    2021-05-13
    计算机网络

    速率

    速率即==数据率==或==数据传输率==或==比特率==。

    比特 1/0 位
    主机在数据信道上传输数据位数的速率。
    单位 b/s, kb/s, Mb/s, Gb/s, Tb/s

    发送端 $\rightarrow$ 接收端 6b/s
    010101

    速率和存储容量的单位换算

    • 速率：1 Tb/s = $10^3$ Gb/s = $10^6$ Mb/s = $10^9$ kb/s = $10^{12}$ b/s
    • 存储容量：1 TB = $2^{10}$ GB = 1024 GB，1 GB = $2^{10}$ MB = 1024 MB, 1 MB = $2^{10}$ KB = 1024 KB

    单位有不同，单位之间的进制也不同，速率是以 b (bit) 为单位，存储容量以 B (Byte) 为单位，1 Byte = 8 bit。

    带宽

    1. 带宽是某个信号的频度宽度，最高频和最低频之差，单位是赫兹（Hz）。
    2. 计算机网络中，带宽表示最高数据率。带宽是在发送端最高的发送率。比如 1 Mb/s 带宽可以 1us 发送 1 bit 数据，2 Mb/s 带宽可以 1us 发送 2 bit 数据。

    吞吐量

    单位时间内通过某个网络（或信道或接口）的数据量。单位也是 b/s …

    吞吐量受带宽和网络额定速率的限制。

    链路上实际的数据率相加。带宽是理想值，吞吐量是实际量。
